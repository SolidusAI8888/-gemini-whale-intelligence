# V18 Daily Brief Redesign

本版把正式邮件报告从长审计报告改成每日巨鲸行动简报。

## 主要变化

- 报告第一部分集中展示所有核心图示和关键结论。
- 正文拆成“商界巨鲸行动”和“政界巨鲸行动”。
- 图示以股票/标的为纲，同一股票的 BUY 和 SELL 在同一个图组对比展示。
- Trump 不再单独成章，统一归入政界巨鲸；Trump 与 Pelosi 名字均高亮。
- 默认交易扫描/展示起点为 `SCAN_START_DATE=2026-01-01`。
- 政治期权排序使用披露金额区间；期权名义金额仅作为敞口备注。
- 删除正文中的关联 SELL 审计、行情基本面新闻补充、非主动/税务/授予审计和超长明细。
- GitHub Actions 增加 Berlin 08:00 运行窗口：夏令时/冬令时均可在柏林 8 点发送，手动触发不受限制。

## 新增/建议变量

```text
SCAN_START_DATE=2026-01-01
LOOKBACK_DAYS=200
```

正式邮件仍使用 SendGrid 相关 Secrets。
