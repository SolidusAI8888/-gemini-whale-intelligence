# V35 — 13F Concentration Top 5 Redesign

本版重做“四、机构巨鲸 13F 持仓雷达”的分析结构，替代 V33 的共识信号表，改为用户更直接需要的三张 Top 5 表：

1. **13F 最新持仓集中度 Top 5**
   - 汇总 Top20 机构巨鲸最新一期 13F 持仓。
   - 按同一股票在这些机构中的合计 13F 市值排序。
   - 展示持有机构数、主要持有机构、合计市值、占最新持仓样本比例。

2. **13F 加仓 / 新建仓集中度 Top 5（最近两期）**
   - 比较每家机构最近两期 13F。
   - 汇总同一股票的正向变化。
   - 新建仓视为从 0 增加到本期持仓。

3. **13F 减仓 / 清仓集中度 Top 5（最近两期）**
   - 比较每家机构最近两期 13F。
   - 汇总同一股票的负向变化绝对值。
   - 清仓/退出视为从上期持仓减少到 0。

## 重要口径

- 13F 是季度末持仓披露，不代表实时交易，也不代表同一天买入/卖出。
- 加仓/减仓是“相邻两期季度末持仓市值变化”，可能同时受到股价变化和持股数量变化影响；在当前项目数据结构下，优先用 13F 市值变化表示集中度。
- 为保证最近两期比较可用，workflow 默认 `INSTITUTIONAL_13F_FILINGS_PER_MANAGER` 改为 `2`。
- 报告读取 13F 时不再受 `SCAN_START_DATE=2026-01-01` 限制，允许读取上一期如 2025-12-31 的 13F。
