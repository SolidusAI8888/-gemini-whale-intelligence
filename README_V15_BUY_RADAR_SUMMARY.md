# V15 Buy Radar + Key Summary

本版在 V14 OGE 诊断基础上继续修复报告语义和第一屏阅读效率。

## 主要变化

1. 新增报告第一屏「关键结论摘要」。
   - 明显主动买入：股票、买入总金额、同期卖出、净额、主要买入巨鲸。
   - 明显减持/卖出：股票、卖出总金额、同期买入、净额、主要卖出巨鲸。
   - 新增关键金额对比图。

2. 「主动买入 Top Signals」重构为「主动买入雷达（P/BUY，按去重买入金额）」。
   - 入选条件：存在 P/BUY 主动买入。
   - 不再在该表显示“方向=SELL”或“净信号=减持/卖出预警”。
   - 新增买入强度、净流向、风险标签、买入金额、卖出金额、净额、买/卖比。

3. 新增「主动买入股票关联 SELL 审计」。
   - 进入主动买入雷达的股票，只要同期存在 SELL，就强制展示 SELL 明细。
   - 解决 TSLA 有卖出记录但未进入全市场卖出 Top N 时无法核对的问题。

4. 保留「减持/卖出预警 Top Signals」，但明确它是全市场卖出金额 Top N。
   - 不代表其它股票没有 SELL。
   - 与主动买入股票相关的 SELL，请优先看关联审计。

5. 图表共同报告人去重。
   - CRM / ValueAct 这类共同报告人不再在图表中视觉放大为多笔经济买入。
   - 明细仍保留原始报告人，便于审计。
