# V37 — 13F Count Concentration Charts + Balanced Details

本版本针对“四、机构巨鲸 13F 持仓雷达”和首页摘要做两类修复。

## 1. 首页摘要新增 13F 三张扫描图

在“一、今日结论总览”中新增：

1. `13F 最新持仓集中度 Top 5（按持有机构数）`
2. `13F 加仓 / 新建仓集中度 Top 5（按加仓机构数）`
3. `13F 减仓 / 清仓集中度 Top 5（按减仓机构数）`

这些图表的“集中度”第一排序不再是金额，而是采取同样行为的机构数量。

## 2. 集中度口径改为机构数量优先

- 最新持仓集中度：至少 2 家机构共同持有同一标的。
- 加仓 / 新建仓集中度：至少 2 家机构在最近两期 13F 中共同加仓或新建仓同一标的。
- 减仓 / 清仓集中度：至少 2 家机构在最近两期 13F 中共同减仓或清仓同一标的。
- 金额仅作为同机构数量下的第二排序参考。

## 3. 机构巨鲸 13F 持仓明细改为均衡展示

旧逻辑：所有 13F 明细按金额全局排序并截断，容易导致 Pershing Square / Tiger Global 等少数大额机构占满明细表。

新逻辑：

- 按机构分组；
- 每家已成功采集的机构只显示最新一期 Top 5 持仓；
- 避免一两家机构占满整张明细表；
- 两期比较仍只用于加仓/减仓集中度分析，不混入持仓明细表。
